# 👑 Rolling Beats: The Soul-Saving Index
### A Billboard Hot 100 Analysis (2000-2024) using Tenacious D's 'Tribute' to tell the story

## 📖 The Project
This project is a "Magazine Editorial" style data analysis. I explored the Billboard Hot 100 data to determine which song is “The Greatest Song in the World” that saved Tenacious D as they performed it for the Devil. 

## 🛠️ The Tech Stack
* **Python** (Pandas, Seaborn, Matplotlib)
* **Jupyter Notebooks**
* **Google BigQuery** (Original data source)
* **GitHub** (Version control and hosting)

## 📊 How to View
1. Open the `rollingbeats_melina.ipynb` file directly here on GitHub to see the charts and the editorial layout.
2. The raw data used for the analysis can be found in the `/data` folder.
